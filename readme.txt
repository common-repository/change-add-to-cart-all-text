== Description ==

The "Change Add to Cart All Text" plugin allows you to easily customize the text displayed on the "Add to Cart" button in your WooCommerce store. You can change the default text to better suit your specific product or brand.

== Features ==

* Customize the "Add to Cart" button text for your WooCommerce products.
* Easy-to-use settings panel to configure the custom text.
* Supports all WooCommerce product types and variations.

== Installation ==

1 Upload the `woo-change-add-to-cart-text` folder to the `/wp-content/plugins/` directory.
2 Activate the plugin through the 'Plugins' menu in WordPress.
3 Go to the "Change Add to Cart All Text" settings page under the WooCommerce menu.
4 Enter your desired custom text in the provided field.
5 Save the changes.

== Frequently Asked Questions ==

= How do I revert to the default "Add to Cart" text? =
To revert back to the default "Add to Cart" text, simply delete the custom text from the settings field and save the changes.
